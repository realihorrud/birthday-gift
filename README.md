# Happy Birthday Natasha ✨

A single-page celebration site built with Next.js and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Add Natasha's photos

Drop up to 8 photos into:

```bash
public/natasha/
```

Use these filenames so the gallery picks them up:

```bash
1.jpg
2.jpg
3.jpg
4.jpg
5.jpg
6.jpg
7.jpg
8.jpg
```

If any file is missing, the page falls back to `public/photo.avif` for that slot.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
